# GenQue SME Web

Standalone frontend package for the Subject Matter Expert panel.

## Run
1. Install Node.js 20+.
2. Open a terminal in this folder.
3. Run:

```powershell
npm install
npm run dev
```

Open the localhost URL shown by Vite.
